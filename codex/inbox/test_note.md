# Markdown Syntax

This is a quick review of the markdown syntax.

## Titles

# h1
## h2
### h3
#### h4
##### h5

``` markdown
# H1
## H2
### H3
#### H4
##### H5
```
    
## Lists

#### Unordered List

* Unordered list item 1
* Unordered list item 2
* Unordered list item 3

``` markdown
* Unordered list item 1
* Unordered list item 2
* Unordered list item 3
```

#### Ordered List

1. Ordered list item
2. Ordered list item
3. Ordered list item

``` markdown
1. Ordered list item
2. Ordered list item
3. Ordered list item
```

## Code

This is a ```inline``` code block.

```
This is a code block
```
