# Markdown Syntax

This is a quick review of the markdown syntax.

## Titles

``` markdown
# H1
## H2
### H3
#### H4
##### H5
```
    
## Lists

``` markdown
* Unordered list item 1
* Unordered list item 2
* Unordered list item 3
    
1. Ordered list item
2. Ordered list item
3. Ordered list item
```
